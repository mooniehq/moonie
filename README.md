# Moonie

## Contributing guidelines

```
npm install && npm prune
npm run dev
```

Configuration in `.env`:
* `DB_USERNAME`
* `DB_PASSWORD`
* `DB_HOST`

VSCode settings (`/.vscode/settings.json`):
```
{
    "editor.tabSize": 2,
    "javascript.validate.enable": false
}
```
